## Inštrukcie

Svoje testovacie súbory a adresáre umiestnite sem do **nového balíka** pomenovaného podľa názvu vašej hry. Napr.:
```bash
sk.tuke.gamestudio.game.mines.FileTest.java
```

## Instructions

Place your unit test files here into a **new package** named after your game. E.g.:
```bash
sk.tuke.gamestudio.game.mines.FileTest.java
```
