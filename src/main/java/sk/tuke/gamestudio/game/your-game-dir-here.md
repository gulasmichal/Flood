## Inštrukcie

Súbory a adresáre vašej hry umiestnite sem do **nového balíka** pomenovaného podľa názvu vašej hry. Napr.:

```bash
sk.tuke.gamestudio.game.mines.Minesweeper.java
sk.tuke.gamestudio.game.mines.core
sk.tuke.gamestudio.game.mines.consoleui
```

## Instructions

Place your game files and directories here into a **new subpackage** named after your game. E.g.:

```bash
sk.tuke.gamestudio.game.mines.Minesweeper.java
sk.tuke.gamestudio.game.mines.core
sk.tuke.gamestudio.game.mines.consoleui
```
